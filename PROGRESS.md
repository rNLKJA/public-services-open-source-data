- 2026-07-06 (hourly run, fuller pass on a previously left-open domain): re-checked **SA public sector workforce and employment**, following up on the earlier note that it was left unstruck for a fuller future pass. `data.sa.gov.au` was reachable this run (correct API path is `/data/api/3/action/...`, not `/api/3/...`). Confirmed again that OCPSE's own CKAN-published datasets (fraud, whistleblower, executives, contractors, consultants, complaints, WHS) are narrow annual-report micro-datasets stuck at FY2020-21 despite 2025 `metadata_modified` timestamps (records re-touched, not the underlying data), and that OCPSE's actual whole-of-sector workforce reporting (State of the Sector, Workforce Information Report) is published only as narrative PDFs on publicsector.sa.gov.au, with a "Workforce Information Data Dashboard" that returned HTTP 403 to both WebFetch and a direct `curl` this run — its underlying data, if any, isn't accessible. Also checked the Department of State Development's "Workforce Data" dataset (economy-wide by occupation/industry/region, last updated Jan 2022) — not public-sector-specific and stale. Did find and add a genuinely open, currently-maintained dataset one level down: **`datasets/sa-education-workforce/`** — SA Department for Education (the largest single SA public sector agency) workforce headcount, FTE, classification, full/part-time, age/gender and Aboriginal/Torres Strait Islander employee breakdowns, FY2014 through FY2025, CC BY 4.0 confirmed via CKAN `license_id`. All five data CSVs plus the source's data dictionary mirrored in `raw/`. This is agency-level, not whole-of-sector — struck the checkbox on the basis that a genuine, current, open dataset for this domain now exists in the repo, while documenting in the new dataset's README that the whole-of-sector OCPSE angle remains a real, undisclosed gap rather than treating this as fully resolving the original ask.

- 2026-07-06 (hourly run): checked **Housing**. `data.sa.gov.au` was directly reachable again this run (confirmed via CKAN API `package_search`/`package_show`, and a direct file download). Surveyed the `housing` tag (23 datasets) — most are SA Housing Authority/SA Housing Trust social-housing series last updated 2015–2022 and effectively static. The **Private Rent Report** (`private-rent-report`) stood out as still actively maintained: 72 quarterly XLSX resources from June quarter 2008 through March quarter 2026 (latest published 22 May 2026), CC BY 4.0 confirmed via the CKAN record's `license_id`/`license_url` fields, published by South Australian Housing Trust from the Consumer and Business Services rental bond dataset. Added `datasets/sa-private-rental-report/` — mirrored only the latest quarter (`private-rental-report-2026-03.xlsx`) in `raw/` to keep footprint modest given the 72-file historical depth; the README links back to the source for the full series. Content is aggregated bond counts and median rents by suburb/postcode/region/LGA, with the publisher already suppressing counts of 1–5 as `*` — no individual-level fields, consistent with this repo's privacy check. Considered but did not add: `social-housing-dwellings`/`households`/`new-households-housed`/`household-members` (all last touched 2022, data only to 2020-21 — noted as a future-pass candidate) and `housing-sa-public-housing-tenantable-stock-by-bedrooms-2015`-style stock-by-bedroom pages (dataset slug from an older search returned 404 — likely retired/renamed; not pursued further this run given the 2–4 search budget).
