- 2026-07-06 (scheduled run): checked **Libraries**. Career context re-checked against memory — Rin's ASO7 SAPOL analytics track is unchanged, so proceeded top-of-list as usual, no reordering needed. Used a two-angle search (data.sa.gov.au CKAN vs. wider web) to survey candidates, then independently re-fetched each promising one directly rather than trusting search summaries. Surprising finding: **"Libraries Board of South Australia Annual Report Data"** (data.sa.gov.au, CC BY 4.0, 8 annual resources through 2023-24) looked like the obvious statewide fit by name, but downloading and reading its actual 2023-24 CSV directly showed it is **not** library usage/membership data — it's the Libraries Board's standard whole-of-government annual-report disclosure template (fraud instances, whistleblower disclosures, executive employment, consultant/contractor spend, complaint categories), the same "narrow annual-report micro-dataset" pattern already documented for OCPSE. Also checked and excluded: City of Adelaide's "City Library Daily Door Count" (CC BY 4.0 but data frozen since 17 June 2018 despite a "weekly" update claim — 8 years stale), "City Library Usage for the 1st year" (CC BY 4.0, no individual data, but a fixed one-off Jan 2014–Feb 2015 snapshot, `update_freq: never`), City of Onkaparinga's "Public Libraries Location Map" (CC BY 4.0 but its only resource is a hyperlink to a council webpage, not structured data), and two discontinued State Library of SA one-off snapshots (Playford library catalogue 2015; Dewey call-number counts, whose own description says it stopped being updated when the compilation method changed). With no genuine current statewide SA-specific dataset available, fell back to a national source per this repo's standing rule: the **Australian Public Libraries Statistical Report 2022-23** (ALIA/NSLA, published October 2024), compiled from data South Australia's own Public Library Services SA supplies annually. Downloaded the actual 15.5MB PDF directly (`read.alia.org.au`, no auth needed) and read it directly rather than trusting a secondary summary — good thing, since a web search result had claimed this edition was CC BY-NC-SA 4.0, which the PDF's own cover page contradicts outright (plain CC BY 4.0, confirmed visually). Extracted South Australia's own FY2022-23 figures from the report's per-jurisdiction comparison tables (collections: 2.91m items; usage/loans: 11.93m; onsite visitors: 7.57m; website visits: 9.17m; program sessions: 49,497; program participants: 714,543) — all aggregate, no individual-level data. Noted as a real limitation that this report's per-state tables don't break out registered-member counts, branch/service-point counts, staff, or expenditure by jurisdiction (national totals only for those), so the "membership" and "branch locations" parts of this domain remain a genuine gap, not silently dropped. A newer 2023-24 edition exists (released October 2025) but is distributed only via the National Library of Australia's Trove viewer, which returned no extractable content or direct download link to either an automated fetch or a follow-up web search — used the 2022-23 edition instead since it could be fully verified by direct download. Added **`datasets/sa-public-library-statistics/`**; also added a CC BY 4.0 attribution line to `LICENSE-DATA.md` for ALIA/NSLA.
